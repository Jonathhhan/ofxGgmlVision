# ofxGgmlVision v1.0.1

## Compatibility

- openFrameworks: `of_v20260406_vs_64_release`
- minimum `ofxGgmlCore`: `1.0.1` / commit `2171e8b`
- `OFXGGML_VISION_VERSION_STRING`: `1.0.1`

## Highlights

- Added independent Vision addon version metadata.
- Exposed version metadata through `ofxGgmlVision.h`.
- Documented the release checklist and release policy for vision workflows.
- Kept vision-specific request helpers out of `ofxGgmlCore`.
- Preserved the root-level `ofxGgmlVisionImageExample` as the first example
  lane.

## Runtime Scope

This release includes public request/result helper types, input validation,
description helpers, a root-level image example skeleton, local validation, and
headless tests.

It does not include a complete CLIP, image embedding, captioning, visual search,
or VLM runtime adapter yet. Those remain roadmap tasks.

- None relative to the new companion-addon line.
