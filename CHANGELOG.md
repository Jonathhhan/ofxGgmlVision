# Changelog

## 1.0.1 - 2026-05-12

- Added independent Vision addon version metadata.
- Exposed version metadata through the public umbrella header.
- Documented the release checklist, release policy, and `v1.0.1` scope.
- Kept CLIP, image embedding, captioning, and VLM runtime adapters as explicit
  future work.

## 1.0.0

- Started `ofxGgmlVision` as the companion addon for CLIP, image embeddings,
  captions, VLM-style image understanding, and visual search workflows on top
  of `ofxGgmlCore`.
- Added the initial request/result helpers, root-level image example skeleton,
  local validation, and headless tests.
